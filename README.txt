Self Playing Pacman Program: README: This program is an extension of a class project done for Stanford’s CS221 – Principles of Artificial Intelligence Class. To play: 1. Using Pacman as a Minimax agent: type:	python pacman.py -p MinimaxAgent 2. Use more efficient Alpha-Beta Pruning type:	python pacman.py -p AlphaBetaAgent -a depth=33. Using ExceptiMax Algorithm type: 	python pacman.py -p ExpectimaxAgent -a depth=3

4. Simply play Pacman: 

type:   python pacman.py
